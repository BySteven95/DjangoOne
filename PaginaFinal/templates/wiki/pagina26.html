NVIDIA GRID™ es la tecnología más avanzada para compartir GPU virtuales (vGPU) en múltiples instancias de escritorios y aplicaciones virtuales. Ahora puedes aprovechar la potencia máxima de la GPU para centros de datos de NVIDIA para brindar una experiencia de gráficos virtuales superior a cualquier dispositivo, en cualquier lugar. La plataforma de NVIDIA GRID ofrece los mejores niveles de rendimiento, versatilidad, administración y seguridad, para brindar una experiencia de usuario apropiada para cualquier flujo de trabajo.

Nvidia ofrece 3 servicios esenciales:

  • Aplicaciones virtuales de NVIDIA GRID

    • Para las organizaciones que implementan XenApp u otras soluciones de Host de servicios de escritorio remoto (RDSH). Diseñada para proporcionar aplicaciones de Windows en PC con el máximo rendimiento.

  • NVIDIA GRID Virtual PC

    • Para usuarios que desean un escritorio virtual, pero que también necesitan una excelente experiencia de usuario para aprovechar los videos de alta definición, los navegadores y las aplicaciones de PC.

  • Estación de trabajo virtual de NVIDIA GRID

    • Para los usuarios que desean usar aplicaciones de gráficos profesionales con el máximo rendimiento en cualquier dispositivo, en cualquier lugar.
